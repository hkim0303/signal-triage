Signal Triage

An AI-assisted operational triage queue: incoming signals get scored for urgency and given a recommended next action. Nothing executes automatically, a human always approves, edits, or dismisses first.

Built by Heewon Kim — Senior Product Manager, personal project

Keyboard shortcuts on the front card of the queue: A approve · E edit & approve · D dismiss

Demo mode Scoring and recommendations below are rule-based, not a live model.
Settings

Live mode calls Claude's API directly from your browser using your own key, via Anthropic's documented "bring your own key" support. Your key is stored only in this browser tab's session storage, it is cleared when you close the tab, and it is sent only to api.anthropic.com, never to any server of mine, because this tool has no backend server at all. Live calls use Claude Sonnet 5 at published rates (as of 2026-07-19: $2 / $10 per million input/output tokens); check Anthropic's current pricing since this changes over time.
